## 🔬 详细技术流程

### 阶段一：自编码器预训练

```mermaid
graph LR
    A1[输入: 单张可见光图像] --> B1[编码器 without CBAM]
    B1 --> C1[特征图]
    C1 --> D1[解码器]
    D1 --> E1[重建图像]
    E1 --> F1[计算损失: L=λL_ssim + L_p]
    F1 --> G1{反向传播}
    G1 --> B1
```

#### 关键技术点

| 项目 | 说明 |
|------|------|
| **输入** | 单张可见光图像 (RGB) |
| **网络结构** | 编码器 + 解码器（无CBAM） |
| **损失函数** | L = λ × L_ssim + L_p |
| **训练目标** | 重构输入图像 |
| **输出权重** | 基础编码器-解码器预训练权重 |

#### 损失函数公式

$$L_{total} = \lambda \cdot L_{SSIM} + L_{pixel}$$

其中：
- $L_{SSIM}$: 结构相似性损失
- $L_{pixel}$: 像素级损失 (MSE/L1)
- $\lambda$: 平衡参数

### 阶段二：CBAM微调

```mermaid
graph LR
    A2[加载预训练权重] --> B2[固定主干权重]
    B2 --> C2[插入CBAM模块]
    C2 --> D2[编码器 with CBAM]
    D2 --> E2[通道注意力 → 空间注意力]
    E2 --> F2[解码器]
    F2 --> G2[重建图像]
    G2 --> H2[仅更新CBAM参数]
    H2 --> I2{反向传播}
    I2 --> C2
```

#### CBAM模块结构

```
输入特征图
    ↓
[通道注意力]
    ├── 全局平均池化
    ├── 全局最大池化
    ├── 共享MLP
    └── Sigmoid激活
    ↓
[空间注意力]
    ├── 通道池化
    ├── 卷积层
    └── Sigmoid激活
    ↓
输出特征图
```

#### 关键技术点

| 项目 | 说明 |
|------|------|
| **加载权重** | 使用阶段一保存的预训练权重 |
| **固定参数** | 编码器和解码器权重冻结 |
| **可训练参数** | 仅CBAM模块参数 |
| **训练目标** | 学习通道和空间注意力 |

## 🏗️ 网络架构详细图

### 阶段一：基础AutoEncoder

```
可见光图像 (H×W×3)
    ↓
┌─────────────────────────┐
│      编码器 (Encoder)     │
│  Conv → DenseBlock → Conv │
└─────────────────────────┘
    ↓
特征图 (H/8 × W/8 × 64)
    ↓
┌─────────────────────────┐
│      解码器 (Decoder)     │
│  Conv → DenseBlock → Conv │
└─────────────────────────┘
    ↓
重建图像 (H×W×3)
```

### 阶段二：集成CBAM

```
可见光图像 (H×W×3)
    ↓
┌─────────────────────────┐
│      编码器 (Encoder)     │
│  Conv → DenseBlock → Conv │
└─────────────────────────┘
    ↓
特征图 (H/8 × W/8 × 64)
    ↓
┌─────────────────────────┐
│    CBAM注意力模块        │
│  通道注意力 + 空间注意力   │
└─────────────────────────┘
    ↓
注意力特征图
    ↓
┌─────────────────────────┐
│      解码器 (Decoder)     │
│  Conv → DenseBlock → Conv │

    ↓
重建图像 (H×W×3)
```

## 📋 训练配置建议

| 阶段 | 批量大小 | 学习率 | Epochs | 备注 |
|------|---------|--------|--------|------|
| 阶段一 | 16-32 | 1e-4 | 50-100 | 重构预训练 |
| 阶段二 | 16-32 | 1e-4 | 20-50 | CBAM微调 |
| 阶段三 | 8-16 | 1e-5 | 30-80 | 融合训练 |

---

## 🎯 关键参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| `λ_ssim` | SSIM损失权重 | 0.8-1.0 |
| `λ_pixel` | 像素损失权重 | 1.0 |
| `lr` | 初始学习率 | 1e-4 ~ 1e-5 |
| `weight_decay` | 权重衰减 | 1e-4 |
| `batch_size` | 批量大小 | 8-32 |
