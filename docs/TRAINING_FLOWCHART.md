```mermaid
graph TB
    subgraph Stage1["🎯 阶段一：自编码器预训练"]
        S1_A[输入: 单张可见光图像] --> S1_B[编码器 without CBAM]
        S1_B --> S1_C[解码器]
        S1_C --> S1_D[计算重建损失 L=λL_ssim+L_p]

    end
    
    subgraph Stage2["🎯 阶段二：CBAM微调"]

        S2_D --> S2_E[保存CBAM权重]
    end
    
    subgraph Stage3["🎯 阶段三：端到端融合"]
        S3_A[输入: 红外-可见光图像对] --> S3_B[编码器+CBAM 特征提取]
        S3_B --> S3_C[红外特征] & S3_D[可见光特征]
```

## 🔬 阶段一：自编码器预训练

```
输入: 单张可见光图像
    ↓
┌──────────────────────────────┐
│     编码器 (Encoder)         │
│  Conv → DenseBlock → Conv   │
└──────────────────────────────┘
    ↓
特征图 (H/8 × W/8 × 64)
    ↓
┌──────────────────────────────┐
│     解码器 (Decoder)         │
│  Conv → DenseBlock → Conv   │
└──────────────────────────────┘
    ↓
重建图像
    ↓
损失函数: L = λ × L_ssim + L_pixel
```

**训练目标**: 重构输入图像，学习基础特征表示

```
红外图像                              可见光图像
    ↓                                    ↓
┌────────────────────────────┐  ┌────────────────────────────┐
│  编码器+CBAM              │  │  编码器+CBAM              │
│  (Encoder+CBAM)           │  │  (Encoder+CBAM)           │
└────────────────────────────┘  └────────────────────────────┘
    ↓                                    ↓
红外特征图                          可见光特征图
    ↓                                    ↓
    └────────────┬───────────────────────┘
                 ↓
    ┌────────────────────────────────────┐
    │         融合层 (Fusion Layer)        │
    │                                      │
    │  策略1: 加法融合                     │
    │    F = feature_ir + feature_vis      │
    │                                      │
    │  策略2: L1-norm融合                  │
    │    energy = |feature_ir| + |feature_vis|
    │    weight = energy / total           │
    │    F = weight × feature_ir + weight × feature_vis
    │                                      │
    └────────────────────────────────────┘
                 ↓
    ┌────────────────────────────┐
    │      解码器 (Decoder)       │
    └────────────────────────────┘
                 ↓
            融合图像
```

## 🔧 损失函数定义

### 阶段一&二：重构损失

$$L_{reconstruction} = \lambda_{ssim} \cdot L_{SSIM} + \lambda_{pixel} \cdot L_{pixel}$$

其中：
- $L_{SSIM}$: 结构相似性损失
- $L_{pixel}$: 像素级损失 (L1/MSE)
- $\lambda_{ssim}$: SSIM权重 (默认: 1.0)
- $\lambda_{pixel}$: 像素权重 (默认: 1.0)

### 阶段三：融合损失

$$L_{fusion} = \lambda_1 \cdot L_1 + \lambda_2 \cdot (1-SSIM) + \lambda_3 \cdot L_{grad} + \lambda_4 \cdot L_{TV}$$

其中：
- $L_1$: L1像素损失
- $SSIM$: 结构相似性
- $L_{grad}$: 梯度损失
- $L_{TV}$: Total Variation损失
